# 学生信息管理系统 python3

本项目是基于 Python Flask 框架、结合 MySQL 数据库、HTML、CSS 和 JavaScript 技术实现的学生信息管理系统。以下是对本系统的详细介绍。

## 技术栈

- 后端：Python Flask
- 数据库：MySQL
- 前端：HTML、CSS、JavaScript

## 功能模块

- 学生信息管理
- 学生信息综合分析

## 系统角色

- 管理员：负责维护学生信息，进行数据分析等操作。

## 运行环境

- Python 环境推荐使用 Pycharm
- 数据库管理工具推荐使用 Navicat

## 部署步骤

1. 配置 Python 开发环境。
2. 安装 Flask 框架。
3. 安装并配置 MySQL 数据库。
4. 根据文档说明运行项目。

## 目录结构

```
project/
│
├── app/             # Flask 应用目录
│   ├── templates/   # HTML 模板文件夹
│   └── static/      # CSS 和 JS 文件夹
│
├── database/        # 数据库相关文件
│
└── src/             # 源代码目录
    └── main.py      # 主程序入口
```
